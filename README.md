# 小红视频下载器 - 前端

## 项目结构

```
frontend/
├── src/                # 源代码目录
│   ├── assets/        # 静态资源（图片等）
│   ├── components/    # 可复用组件
│   ├── pages/        # 页面文件
│   └── styles/       # 样式文件
```

## 页面说明

- `home.html`: 主页，包含视频下载功能
- `about.html`: 关于页面
- `faq.html`: 常见问题解答
- `feedback.html`: 用户反馈
- `blog.html`: 博客页面

## 开发说明

1. 所有页面文件都位于 `src/pages` 目录下
2. 静态资源（图片等）存放在 `src/assets` 目录下
3. 可复用的组件放在 `src/components` 目录下
4. 样式文件统一放在 `src/styles` 目录下

## 使用说明

直接在浏览器中打开 `src/pages/home.html` 即可使用，或使用本地服务器托管前端文件。
